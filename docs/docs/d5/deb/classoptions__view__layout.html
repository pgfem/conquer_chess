Detailed Description

The layout of the options view

x x x x x 1 2 3 4 5

+---------------------------------------------------—+ | | | +------------------—+-------------------------—+ | y1 | | game speed | | | | +------------------—+-------------------------—+ | y2 | | music volume | | | | +------------------—+-------------------------—+ | y3 | | sound effects | | | | +------------------—+-------------------------—+ | y4 | | starting pos | | | | +------------------—+-------------------------—+ | y5 | | | +----------—+ | y6 | | | |

BOARD

| +----------—+ | y7 | | | +---------------------------------------—+ | y8 | | controls | | | +---------------------------------------—+ | y9 | | left_controls | right_controls | | | +---------------------------------------—+ | y10 | | +---------------------------------------------------—+

Top panel must be 400 pixels at least (i.e. with current font size) Bottom panel must be 400 pixels at least (i.e. with current font size)
